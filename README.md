# monitor-5070ti

Скрипт для мониторинга доступности RTX 5070 Ti в выбранных магазинах и отправки уведомлений.

## Быстрый старт

1. Установите Python 3.11+.
2. Запустите `install_windows.bat` для установки зависимостей.
3. Скопируйте `.env.example` в `.env` и заполните значения.
4. Запустите `run_monitor.bat` или `python -m monitor_5070ti.main`.
