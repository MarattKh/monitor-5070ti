# TASK_FOR_CODEX

Мини-проект мониторинга источников по RTX 5070 Ti с поддержкой:
- конфигурируемых источников;
- фильтрации по ключевым словам;
- экспортов Markdown/JSON/CSV;
- генерации `latest_ai_prompt.md`;
- уведомлений в Telegram и e-mail;
- безопасной работы через переменные окружения.

Если источник нестабилен для requests, должна быть пометка `requires_playwright=true`.
